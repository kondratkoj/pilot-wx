# pilot-wx
A METAR, TAF, and NOTAM decoder for user-defined airports.

Right now, this project only decodes METARs. Later on I'll implement TAF and then NOTAM decoding and filtering. 

The aim is to make a weather data aggregate for fun. The idea is that you tell it your base airport, frequently visited airports, and timeline, and the program will give you all of the pertinent METARs, TAFs, and NOTAMs for your defined airports.
